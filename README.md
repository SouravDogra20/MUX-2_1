2:1 Multiplexer Design using Verilog HDL
Project Description

This project implements a 2:1 Multiplexer (MUX) using structural modeling in Verilog HDL.
The design is created using basic logic gates (AND, OR, NOT) and simulated in Xilinx Vivado.

A 2:1 Multiplexer selects one of the two input signals based on the select line s and forwards it to the output y.

Structural Design

The circuit is designed using:

2 AND gates
1 OR gate
1 NOT gate

Simulation Result

The waveform verifies correct MUX operation:

When s = 0, output follows in1
When s = 1, output follows in2
